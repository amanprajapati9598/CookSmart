# 🍳 CookSmart

CookSmart is an intelligent, AI-powered food recipe application designed to make cooking easier, interactive, and fun. Built with a modern **Flutter** frontend and a robust **PHP/MySQL** backend, it leverages **Google Gemini AI** to provide smart recipe recommendations.

## ✨ Key Features

- **🤖 AI Recipe Recommendations:** Get personalized meal ideas powered by Google Gemini based on your preferences.
- **🥘 Smart Pantry Tracker:** Keep track of the ingredients you have at home and find recipes that match.
- **📅 Meal Planner:** Plan your daily meals and track your nutritional intake effortlessly.
- **🌐 Community Feed:** Share your cooking experiences, post photos, and discover what others are cooking.
- **🛒 Grocery List:** Automatically generate shopping lists from recipes you want to try.
- **📱 Cross-Platform:** Beautiful, responsive UI that works smoothly on Android and iOS devices.

## 🛠️ Technology Stack

**Frontend:**
- Flutter (Dart)
- Provider (State Management)

**Backend:**
- PHP
- MySQL Database

**Integrations:**
- Google Gemini AI API (for Smart Recommendations)
- Razorpay API (for Premium Subscriptions)
- Spoonacular API / MealDB API (for Recipe Data)

## 🚀 Getting Started

### Prerequisites
- [Flutter SDK](https://flutter.dev/docs/get-started/install)
- [XAMPP](https://www.apachefriends.org/index.html) (for PHP and MySQL local server)

### Installation Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/amanprajapati9598/CookSmart.git
   ```

2. **Backend Setup:**
   - Move the project to your `htdocs` directory (if using XAMPP).
   - Import the `recipe_db.sql` database file into your phpMyAdmin.
   - Start Apache and MySQL in the XAMPP Control Panel.

3. **Frontend Setup:**
   - Navigate to the frontend directory:
     ```bash
     cd frontend/food_app_flutter
     ```
   - Install dependencies:
     ```bash
     flutter pub get
     ```
   - Run the app:
     ```bash
     flutter run
     ```
