# ROLES AND RESPONSIBILITIES

We, (**Aman Prajapati & Saumya Gupta**) take full responsibility for completing the project **CookSmart - Food Recipe Recommendation Application**. We have developed the complete system from scratch. All tasks and responsibilities were completed by us within the given time period with proper planning and teamwork.

Both of us worked together in all stages of the project such as planning, designing, development, testing, and documentation. Some modules were mainly handled by one person based on strength, but overall work was done with mutual support and discussion.

**Module Responsibilities:**
- **Frontend Module:** Frontend design and user interface were developed by Aman, assisted by Saumya.
- **Backend Module:** Backend logic, API development, and server-side implementation were mainly done by Saumya, with discussion and support from Aman.
- **Database Module:** Database design, schema creation, and configuration were mainly handled by Aman, with integration support from Saumya.
- **Authentication Module:** This module allows users to register and login securely. Users can access their saved recipes, pantry data, and meal plans using their credentials.
- **AI Recommendation Module:** This module integrates Google Gemini AI to generate personalized recipes, provide cooking instructions, and calculate nutritional values based on available ingredients.
- **Pantry & Meal Planner Module:** This module manages the user's kitchen inventory, structured tracking of available ingredients, and enables calendar-based meal planning.

# ABSTRACT

**CookSmart** is an innovative AI-powered mobile application designed to simplify kitchen management and meal planning. The platform provides personalized recipe recommendations based on the ingredients available in the user's pantry. By combining nutritional monitoring with inventory management, the system delivers a complete and efficient solution for maintaining a healthy lifestyle. Users can seamlessly track their pantry, schedule meals, and receive step-by-step cooking guidance using Google Gemini AI integration. The application features a modern, responsive design built on Flutter and a robust backend architecture using PHP and MySQL, ensuring an engaging and seamless experience. CookSmart bridges the gap between traditional cooking and smart technology, empowering users to reduce food waste, save time, and make better dietary choices.

**Keywords:** AI-powered Recommendation, Pantry Management, Meal Planner, Nutrition Tracking, Google Gemini, Smart Kitchen, Mobile Application, Food Waste Reduction.
